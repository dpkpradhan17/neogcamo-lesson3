CSS Box Model

Each element of HTML has itâ€™s own box. Now a box can have different display properties like block, inline or inline-block. The CSS Box Model applies to these boxes of HTML elements. They describe the properties such as padding, border and margin around the content. By declaring properties of box one can set the behaviour of the box that how it should be displayed.

Content: It is where all the text, images or any other elements reside.
Padding: It is a transparent space assigned by the user around the content box.
Border: It is a border around the content and padding. It can be of many styles like solid, dashed or dotted etc.
Margin: It is a transparent space assigned around the border, padding and content.
