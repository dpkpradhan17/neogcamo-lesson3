How a website works ?

HTML || CSS || Javascript

HTML
  • HTML means Hyper Text Markup Language. HTML consists of tags and elements.
  • Using these tags and elements we provide a basic structure to a website.
  • The fundamental building block of any page.
CSS
  • CSS stands for cascading stylesheets. As the name suggests it is used to give a proper style to your website so that it looks good whenever someone visits the site.
  • Primarily handles look and feel of a web page.
  • Make web-pages responsive so that user can view it on different devices.
Javascript
  • Javascript is a scripting language which provides a way to add functions to those elements which were added to the website during HTML coding.
  • It handles all the functions of a webpage.
  • A user can use javascript for more interactive web-page.

Anyone can make a website by having basic knowledge about these three things. If you inspect this page you can see that even this page is made up of simple program. There is no complex coding.
